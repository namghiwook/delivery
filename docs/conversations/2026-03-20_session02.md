## 1. 점주가 가맹하지 않는 이유 (질문에서 시작)

**사용자 질문:** 지자체 배달앱 가맹점이 턱없이 없는 이유는?

**핵심 답변:**
- "공짜"가 아님 - 메뉴 등록, 태블릿 관리, 시스템 학습 등 시간/수고 비용
- 핵심: 점주는 "얼마를 내느냐"가 아니라 **"얼마를 벌 수 있느냐"**로 판단
- 배민 입점 = 2,000만 소비자에 노출, 지자체 앱 = 수천 명에 노출
- 주문이 안 오면 태블릿 꺼놓고 방치 → 소비자 경험 나빠짐 → 악순환

**교훈:** "수수료 0%라서 점주가 올 것이다"는 환상. 주문이 와야 점주가 옴.

## 4. 땡겨요 성공의 진짜 이유 (핵심 발견)

**땡겨요는 지자체가 만든 앱이 아니라, 신한은행이 만든 민간 앱이었음**

- 2022년 신한은행이 금융위 승인 받아 출시
- 서울시 '서울배달플러스' 단독 운영사로 선정 (2025년~)
- **서울시(정책/예산/인센티브) + 신한은행(플랫폼/기술/운영)의 민관협력 모델**

신한은행이 배달앱 하는 이유: 비금융 데이터 확보, 금융상품 연계, ESG 경영

**땡겨요 vs 배달특급(경기도) 핵심 차이:**
- 기술: 신한은행 IT vs 중소 SI 업체
- 자금: 신한금융 + 서울시 35억 + 상품권 250억 vs 경기도 50억(감소 추세)
- 소비자 혜택: 최대 30% 할인 vs 제한적
- 배달: 바로고 협업(배달비 900원) vs 가게 직접 배달
- 지속 동기: 신한은행 자체 사업 동기 vs 정치적 동력(지사 교체로 약화)

**한 문장 결론:** "지자체가 직접 플랫폼을 만든 것(배달특급) vs 대기업급 민간 역량에 정책 인센티브를 얹은 것(땡겨요)의 차이"

**공공배달앱 전멸 현황:** 코로나 때 30개 → 현재 15개 미만. 인구 100만 미만 지역 거의 전멸.

## 6. 당근 배달 미진출 근거 검증

**"배달 안 한다"는 경영진 직접 발언이 아닌 기자 해석이었음**
- 실제 당근 측 발언: "정식 서비스 출시 여부는 정해지지 않았다"
- 2021년 '당근배송' 송파구 시범 운영 이력 있음
- CJ대한통운과 물류 MOU 체결 (2025년)
- 당근미니로 외부 서비스 연동 구조 존재
- 결론: 배달 진출을 배제한 것이 아니라 단계적 확장 중

**당근 협업 가능성:** 논의했으나, "이상한 길로 빠지는 것 같다"는 사용자 판단으로 하이퍼로컬 본론으로 복귀

## 7. 삼각 딜레마 발견 (핵심!)

### 배달비 구조의 근본 문제

소비자가 배민/쿠팡이츠에서 안 내던 배달비를 우리 플랫폼에서 내야 하는 문제:

```
쿠팡이츠: 음식 15,000원 + 배달비 0원 = 15,000원
우리:     음식 14,250원 + 배달비 3,000원 = 17,250원 (더 비쌈!)
```

### 수수료 절감으로 배달비를 메울 수 있나?

**수학적으로 불가능:**
- 배민 수수료 6.8% × 주문 20,000원 = 1,360원 (최대 절감 가능액)
- 배달비 = 3,000원
- 1,360원 < 3,000원 → **수수료를 0%로 해도 배달비를 커버 못함**
- 커버 가능한 최소 주문금액: 44,118원 (평균 2만원 대비 2배 이상)

**결론:** 수수료율(%)로 절감하는 금액이, 고정액(원)인 배달비보다 구조적으로 작음

### 삼각 딜레마
```
점주 수수료 낮추면 → 배달비 메울 재원 없음
배달비를 소비자에게 받으면 → 배민/쿠팡보다 비싸짐
배달비를 안 받으면 → 라이더 비용(건당 4,000원)을 누가 내나
```

## 8. 해결책: 공공보조금 + 협동조합 모델

### 땡겨요가 삼각 딜레마를 푼 방법: 세금

서울시 배달전용상품권 250억원(15% 할인 판매) + 페이백 10% + 쿠폰 5% = 최대 30% 소비자 할인

### 이걸 하이퍼로컬 LoCo Co-Op에 적용하면?

```
[동네 점주 협동조합] + [구청 배달상품권 지원] + [우리 기술/플랫폼]
```

**구청 설득 논리:**
- 땡겨요: 수익이 신한은행으로 → 우리: 수익이 동네 점주에게 환원
- "우리 구 소상공인이 직접 소유한 배달 협동조합 지원" = 땡겨요보다 명분 강함
- 기존 지역사랑상품권 인프라에 배달 연동 추가 = 새 예산 불필요

### 리스크 논의 (사용자 제기)

| 리스크 | 사용자 우려 | 완화 장치 |
|--------|-----------|----------|
| 점주 무임승차 | 보조금 받고 가격 안 내림 | 보조금을 소비자에게 직접 지급 (상품권) |
| 단체 협상력 남용 | 협동조합이 구청에 더 많은 보조금 압박 | 다년 협약 + 성과 연동 |
| 가격 슬금슬금 인상 | 초기만 협조, 이후 인상 | 매장가 모니터링 + 가격 인상 시 노출 하락 |
| 갑을 역전 (가장 위험) | 협동조합이 갑, 우리가 을 | 운영 대행이 아닌 **공동 설립/기술 지분 참여** (30%) |

### 근본 한계
- 보조금 의존 모델 = 보조금 끊기면 끝
- 땡겨요도 같은 리스크를 안고 있음
