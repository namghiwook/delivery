# 당근 포장주문 & 땡겨요 성공 분석

> 작성일: 2026-03-20
> 웹 검색 기반 최신 데이터 포함

---

## 1. 당근 포장주문

### 핵심: 배달앱 경쟁자가 아니라 "포장" 시장의 플레이어

- 가맹점 수: 비공개 (비즈프로필 200만 중 일부). 배민 30~40만 대비 아직 적음
- **포장 전용** - 배달 서비스 제공 안 함, "배달 진출 계획 전혀 없다"고 명시
- 수수료 0%, 배달비 0원(소비자가 직접 픽업)

### 소비자가 쓰는 진짜 이유

**배민이 2025년 4월부터 포장 주문에도 6.8% 수수료를 부과하기 시작**
→ 점주가 배민 포장을 해지하고 당근으로 전환
→ 당근은 수수료 0%이므로 점주가 매장가 그대로 또는 더 낮게 설정
→ 소비자: 같은 메뉴가 당근에서 더 싸거나 같은 가격 + 배달비 0원(픽업)

| 플랫폼 | 포장 수수료 |
|--------|-----------|
| 당근 | **0%** |
| 배민 | **6.8%** (2025.4~ 전면 유료화) |
| 쿠팡이츠 | 2026.3까지 무료 (이후 미정) |

### 결론: 당근은 우리의 직접 경쟁자가 아님
- 배달을 안 하므로 배달 시장과는 별개
- 다만 "수수료 0% + 기존 사용자 풀"의 위력을 보여주는 참고 사례

## 2. 땡겨요 - 가장 중요한 발견

### 핵심: 땡겨요는 지자체가 만든 앱이 아니라, **신한은행이 만든 민간 앱**이다

- 2022년 **신한은행**이 금융위원회 승인을 받아 출시
- 기술: 신한은행 IT + AI 스타트업 '미식의 시대'
- 2025년부터 서울시 공공배달앱 '서울배달플러스' 단독 운영사로 선정
- **서울시(정책/예산/인센티브) + 신한은행(플랫폼/기술/운영)의 민관협력 모델**

### 신한은행이 배달앱을 하는 이유
- **비금융 데이터 확보**: 가맹점 매출 데이터 → 맞춤 금융상품 설계
- **금융 연계**: 당일 정산, 소상공인 대출 등 금융상품 탑재
- **ESG 경영**: '착한 배달앱' 브랜드
- SOL앱에서 바로 이용 가능 → 기존 금융 고객 유입

### 성공 공식

```
신한은행의 대기업급 기술력/자금력
  + 서울시 950만 인구 (시장 규모)
  + 서울시 예산 35억원 + 배달상품권 250억원
  + 소비자 최대 30% 할인 (상품권 15% + 페이백 10% + 쿠폰 5%)
  + 수수료 2% (배민 6.8~11.2% 대비)
  + 바로고 협업 자체배달 (배달비 900원)
  + 서울사랑상품권 결제 38% + 온누리상품권 12%
  = 2025년 매출 1,544억원, 점유율 7.5%
```

### 2024→2025 폭발 성장

| 지표 | 2024년 | 2025년 | 성장 |
|------|--------|--------|------|
| 거래액 | 423억 | **1,544억** | **+265%** |
| MAU | 99만 | **329만** | **+232%** |
| 점유율 | 2.34% | **7.5%** | **3.2배** |

## 3. 배달특급(경기도) 위축 원인

| 요인 | 상세 |
|------|------|
| 운영 주체 | 경기도주식회사(공기업) - 기술력 한계 |
| 지사 교체 | 이재명→김동연, 예산 80억→50억 감소 |
| 배달 인프라 | 자체 배달 없음, 가게 직접 배달 의존 |
| 광역 지역 | 경기도 넓어서 인구 밀도 낮음 |
| 누적 적자 | 28.6억원 (2021~2024) |
| 악순환 | 소비자↓ → 점주↓ → 가맹점↓ → 소비자↓ |

거래액: 2022년 1,310억(정점) → 2024년 556억(-57%) → 2025년 1,046억(반등)

## 4. 공공배달앱 전멸 현황

코로나 당시 30개 → 현재 **15개 미만** 운영

| 중단된 앱 | 지역 | 중단 시점 |
|----------|------|----------|
| 일단시켜 | 강원도 | 2024년 말 |
| 동백통 | 부산 | 2024년 5월 |
| 누비고 | 창원 | 2024년 11월 |
| 배달의진주 | 진주 | 2024년 11월 |
| 배달올거제 | 거제 | 2022년 12월 |

**공통 실패 원인:**
1. 인지도 부족 (점주 애로 1위: 42.5%)
2. 기술력 격차 (민간 1,200~1,700명 vs 공공 두자릿수 인력)
3. 배달 인프라 부재
4. 인구 100만 미만 지역은 거의 전멸
5. 정권 교체 시 예산/동력 상실

## 5. 핵심 교훈

### 한 문장

> **"땡겨요가 되고 배달특급이 안 된 핵심 차이: 지자체가 직접 플랫폼을 만든 것(배달특급) vs 대기업급 민간 역량에 정책 인센티브를 얹은 것(땡겨요)"**

### 우리 프로젝트 시사점

1. **민관협력 모델이 가장 현실적** - 기술은 민간, 인센티브는 공공
2. **소비자에게 확실한 경제적 혜택 필수** - 땡겨요의 최대 30% 할인
3. **인구 100만+ 시장에서 시작** - 소규모 지역은 임계질량 불가
4. **배달 인프라 필수** - 가게 직접 배달은 한계, 배달대행사 연동 최소
5. **운영 주체의 자체 사업 동기** - 신한은행은 데이터/금융 수익이라는 독자 동기
