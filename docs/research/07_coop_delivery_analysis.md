# 생협/소비자 협동조합의 배달 서비스 심층 조사

> 작성일: 2026-03-23
> 웹 검색 기반 조사

---

## 핵심 발견

### 한국 생협은 배달 문제를 건드리지 않았다

- 한살림(84만 세대), 아이쿱(31만), 두레(24만) 어디도 음식 배달 시도 없음
- 기존 사업은 모두 "식재료 정기 배송" (주 1~2회, D-2 주문 마감)
- 배달앱 수수료 문제에 대한 공식 입장도 없음
- "자발적 불편함"을 가치로 표방 → 즉시 배달과 가치관 충돌

### 해외에서 가장 참고할 모델: 영국 Co-op

- Deliveroo, Uber Eats, Just Eat **3곳 모두와 제휴**
- 자체 배달앱 안 만들고 **화이트라벨 배달 제휴**
- 2025년 퀵커머스 점유율 30% 목표
- 20분 내 배달 가능

### CoopCycle (유럽 배달원 협동조합)

- 12개국 70개 협동조합, 200~300명 배달원
- 오픈소스 배달 소프트웨어 공동 개발
- 규모는 작지만 "협동조합이 배달 플랫폼을 운영할 수 있다"는 개념 증명

### 일본 생협

- 조합원 2,960만명, 택배 매출 17.5조원 (매장보다 큼)
- 한반(班) 시스템: 3~5명 이웃이 공동 주문/수령
- 하지만 이것도 "주 1회 정기 배송"이지 즉시 배달이 아님

## 한국 생협 활용의 구조적 장벽

| 장벽 | 상세 |
|------|------|
| **법적 제약** | 소비자생활협동조합법: 비조합원 이용 원칙적 불가 |
| **물류 부적합** | 대형 탑차 주 1~2회 → 오토바이 실시간과 완전히 다른 인프라 |
| **문화적 괴리** | "자발적 불편함" ↔ 즉시 배달 편의성 |
| **기술 역량** | 실시간 매칭/GPS/결제 시스템 개발 역량 부족 |
| **의사결정** | 협동조합 민주적 구조 → 빠른 피봇 불리 |
| **재무 보수성** | 대규모 적자 감수 투자와 충돌 |
