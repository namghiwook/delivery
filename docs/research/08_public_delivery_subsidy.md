# 공공배달앱 보조금 추이 조사

> 작성일: 2026-03-23
> 웹 검색 기반 조사

---

## 핵심 발견: 중앙정부 보조금이 2026년 0원으로 삭감

2025년 650억원을 투입했던 농림축산식품부 국비가 2026년 예산안에 미반영.
공공배달앱 성장의 핵심 재원이 사라지면서, 지자체 자체 예산에만 의존하는 구조로 전환.

---

## 1. 중앙정부 (농림축산식품부)

| 연도 | 예산 | 비고 |
|------|------|------|
| ~2024년 | 0원 | 공공배달앱 별도 지원 없음 |
| **2025년** | **650억원** | 추가경정예산으로 신규 편성 |
| **2026년** | **0원** | 예산안에 미반영 (요청했으나 불가) |

### 2025년 650억원 사업 내용
- 정식 명칭: '공공배달앱 지원사업'
- 전국 12개 공공배달앱(땡겨요, 먹깨비, 배달특급 등)을 통해 지원
- 2만원 이상 주문 시 **5,000원 소비쿠폰** 지급
- 효과: 외식업체 수수료 절감 약 207억원, 소비자 쿠폰 약 303억원

### 2026년 0원의 배경
- 농식품부 관계자: "예산 편성 과정에서 요청했지만 반영되지 않은 상황"
- 출처: 아시아투데이 (2025.10.11)

---

## 2. 서울시 (땡겨요 = 서울배달플러스)

| 연도 | 예산 | 내역 |
|------|------|------|
| 2024년 | **1억 800만원** | 초기 단계 |
| 2025년 | **25억원** | 본격 확대 |
| 2026년 | **35억원** | 홍보비 3억 + 자치구 배달상품권 32억 |

### 추가: 배달전용상품권
- 2026년 상반기: 22개 자치구에서 **250억원** 규모 발행
- 15% 할인 판매 (서울페이+ 경유)
- 실질 보조금: 250억 × 15% = **약 37.5억원**

### 소비자 혜택 구조
- 배달전용상품권 선할인: 15%
- 페이백 (결제금액 환급): 10%
- 땡겨요 자체 쿠폰: 5%
- **최대 30% 할인**

## 3. 경기도

| 연도 | 통큰세일 예산 | 배달특급 별도 | 비고 |
|------|-------------|-------------|------|
| 2024년 | **40억원** | - | 시작 |
| 2025년 상반기 | **70억원** | - | |
| 2025년 하반기 | **50억원** (추경 20억 포함) | - | 연간 합계 120억 |
| 2026년 상반기 | **70억원** | - | 하반기 별도 편성 예정 |

### '2026 경기 살리기 통큰세일' 세부
- 기간: 2026.3.20~3.29 (열흘)
- 대상: 배달특급, 땡겨요, 먹깨비 (공공배달앱 3사)
- 쿠폰: 2만원 이상 주문 시 **건당 5,000원 할인**, 횟수 제한 없음
- 규모: 도내 500여 개 상권, 8만여 개 점포 참여

### 배달특급 운영비 (별도)
- 경기도주식회사 출자금/운영비: 2021년 기준 약 137억원
- 예산 추이: 2022년 80억 → 2023년 60억 → 2024년 62억 → 2025년 50억 (감소 추세)
- 누적 적자: 28.6억원 (2021~2024)
- 2025년: 5년 만에 흑자 전환 (1,100만원)

## 5. 전국 보조금 합산 추이

| 구분 | 2025년 | 2026년 |
|------|--------|--------|
| **국비 (농식품부)** | **650억원** | **0원** |
| 서울시 | 25억 + α | 35억 + 상품권 37.5억 |
| 경기도 | 120억 + 배달특급 50억 | 70억~(상반기) + 배달특급 |
| 기타 지자체 | 수십억 추정 | 수억원 수준 |
| **합계 (추정)** | **약 900~1,000억원** | **약 200~300억원** |

**2025년 대비 2026년 보조금 규모가 약 1/3~1/4로 급감.**
핵심 원인: 국비 650억원 삭감.

## 6. 재원 구조

```
[중앙정부 - 농림축산식품부]
  → 2025년: 추경 650억 (전국 공공배달앱 쿠폰)
  → 2026년: 0원 (삭감)

[광역지자체 - 서울시, 경기도 등]
  → 자체 예산에서 편성 (서울 35억, 경기 70억+)
  → 지역사랑상품권/배달상품권 발행 (서울 250억 규모)

[기초지자체 - 구/군/시]
  → 자치구별 배달상품권 자체 발행
  → 규모 소액 (수천만~수억원)
```

## 7. 시사점

### 보조금 의존 모델의 구조적 취약성이 현실화
- 2025년 국비 650억으로 공공배달앱 "역대 최대 성장" 달성
- 2026년 국비 0원 → 성장 동력 급감 예상
- 시장 왜곡 우려 비판도 제기됨 (전자신문 보도)

### 지자체별 편차 극심
- 서울시(35억+상품권 250억) vs 부산시(5천만원) → 50배 이상 차이
- 인구 규모와 재정 자립도에 따라 공공배달앱 지속 가능성 결정

### 정치적 변동에 극도로 취약
- 경기도: 지사 교체(이재명→김동연) 후 배달특급 예산 80억→50억 감소
- 중앙정부: 1년 만에 650억→0원
- "정권에 따라 생사가 갈리는 구조"
