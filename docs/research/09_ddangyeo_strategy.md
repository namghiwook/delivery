# 땡겨요의 "공공배달앱" 전략 분석

> 작성일: 2026-03-23

---

## 핵심: 민간 앱이 어떻게 공공 타이틀과 국비를 받았나

땡겨요는 신한은행(시가총액 20조원)이 만든 **민간 배달앱**이지만,
"공공배달앱"으로 지정받아 국비 보조금까지 수혜받는 구조를 만들었다.

---

## 3단계 전략

### 1단계: "착한 배달앱" 명분 확보 (2022년)

- 신한은행이 금융위원회 '부수업무' 승인을 받아 땡겨요 출시
- 수수료 2%, 광고비 0원 → "소상공인 상생" 포지셔닝
- 자체 사업 동기: 가맹점 매출 데이터 → 금융상품 설계, 대출 연계

### 2단계: 지자체에서 "공공" 타이틀 획득 (2025년)

- 서울시가 공공배달앱 운영사 공모
- 땡겨요가 단독 운영사로 선정 → **"서울배달플러스 = 땡겨요"**
- 이 시점부터 민간 앱이 **공공 배달앱**이라는 지위 획득
- 이후 전국 51개 지자체와 협약 확대

### 3단계: 국비 보조금 수혜 (2025년)

- 농림축산식품부 '공공배달앱 지원사업' 650억원 편성
- 대상: "수수료가 낮은 공공배달앱"
- 땡겨요는 서울시 지정 공공배달앱 → 당연히 수혜 대상
- 보조금으로 소비자 할인(최대 30%) → 점유율 2.34% → 7.5% 급성장

---

## 신한은행의 실제 수익 구조

```
[세금/보조금이 하는 일]
  소비자 할인 재원 → 사용자 확보 → 플랫폼 성장

[신한은행이 가져가는 것]
  1. 가맹점 매출 데이터 → 맞춤 금융상품 설계
  2. 소상공인 대출 (매출 데이터 기반 신용평가)
  3. 당일정산 서비스 → 선정산 수수료
  4. SOL앱 연동 → 금융 고객 유입
  5. 수수료 2% (거래액 1,544억 기준 연 30억+)
  6. ESG 브랜드 이미지
```

---

## 구조 요약

```
신한은행이 수수료 2%로 설정 (배민 7.8% 대비 낮음)
  → "소상공인 상생"이라는 명분 확보
  → 서울시가 "공공배달앱" 타이틀 부여
  → 국비(농식품부 650억) 보조금 수혜 자격 획득
  → 보조금으로 소비자 할인 → 점유율 급성장
  → 성장한 플랫폼에서 금융 데이터/대출 수익 확보
  → 신한은행 이득
```

---

## "공공" 타이틀의 본질

- 기술적 자격 요건이 아니라 **지자체가 지정해주는 것**
- 수수료가 낮고 소상공인 상생 명분이 있으면 받을 수 있음
- 한번 공공 타이틀을 받으면 국비/지방비 보조금 접근 가능
- **신한은행급 로비력/브랜드가 없어도** 지자체 협약은 가능 (현재 51개 지자체 협약)

---

## 비판과 논란

- **"세금으로 대기업 플랫폼을 키워주는 꼴"**이라는 비판 가능
- 전자신문: "재정 보조에 힘입은 성장, 시장 왜곡 우려"
- 2026년 국비 0원 삭감 → 정부도 지속가능성에 의문을 가진 것으로 해석 가능
- 보조금 없이 소비자를 유지할 수 있는지가 땡겨요의 진짜 시험

---

## 우리 프로젝트 시사점

### 참고할 점
- 낮은 수수료 → 상생 명분 → 공공 타이틀 → 보조금 접근은 **검증된 경로**
- 지자체 협약은 대기업이 아니어도 가능 (소규모 스타트업도 배달특급, 먹깨비 등 운영)
- "공공 타이틀"은 초기 부트스트랩에 유효한 전략

### 주의할 점
- 보조금 의존 구조는 **1년 만에 뒤집힐 수 있음** (650억→0원 사례)
- 보조금은 성장 가속용으로만, **자체 수익 구조가 반드시 병행** 필요
- 신한은행은 금융 수익이라는 자체 동기가 있었음 → 우리도 보조금 외 독자 동기 필요
