# 종합 시스템 제안서: "동네배달"

> 작성일: 2026-03-19
> 4개 심층 조사 결과를 교차 분석하여 도출한 시스템 설계안

---

## 조사 결과 교차 분석에서 나온 핵심 통찰

### 통찰 1: 소비자는 "수수료 0%"에 관심 없다
- 소비자 선택 기준 1순위는 **가맹점 다양성**(55~65%)
- "외국 자본" 이슈로 실제 전환하는 소비자는 **3~5%**에 불과
- 소비자는 **자기가 직접 체감하는 혜택**(가격, 속도, 편의)이 있어야 앱을 바꿈

### 통찰 2: 점주에게 수수료 절감은 순이익의 폭발적 증가
- 점주 순이익률 5~10%인 상황에서, 수수료 6.8% 절감 = **순이익 50~136% 증가**
- 점주의 플랫폼 총비용은 수수료+광고+배달대행 합쳐 매출의 **13~20%**
- 수수료 0~2%면 "반드시 써봐야 한다"는 강한 동기, **3%가 심리적 최대선**

### 통찰 3: 점주를 영업사원으로 만들 수 있다
- 수수료가 충분히 낮으면 점주가 **자발적으로** 소비자에게 우리 플랫폼을 홍보
- "이 앱으로 주문하면 더 저렴합니다" → 점주가 전단지 동봉, 매장 안내
- **소비자 확보 비용(CAC)을 점주가 대신 지불**하는 구조

### 통찰 4: 라이더는 콜 안정성 > 건당 단가
- 콜이 꾸준히 오는 플랫폼에 머묾
- 배차 알고리즘 투명성이 가장 큰 차별화 요소
- 초기에는 **배달대행사 API 연동**이 현실적

### 통찰 5: 하이퍼로컬 + 공동주문이 가장 유망
- 1개 동네에 집중해 가맹점 밀도를 확보하면 소비자 전환의 1순위(가맹점 다양성)를 해결
- 공동주문(3명 모이면 배달비 0원)은 **카카오톡 바이럴** 내장
- 도보/자전거 배달로 배달비 0~1,500원 → 기존 플랫폼 대비 압도적 가격 경쟁력
- 배민이 **구조적으로 따라하기 어려움** (그들은 전국 단위 효율화에 최적화)

### 통찰 6: "레이더 아래로 날기" 전략
- 배민/쿠팡이츠가 대응하는 임계점: 전국 점유율 3~5%
- 그 전까지 **특정 동네에서 전환 비용(switching cost)을 쌓아야** 함
- 한 동네에서 점유율 30~40% 달성이 현실적 목표

## 제안 시스템: 3가지 후보

### 후보 A: "하이퍼로컬 커뮤니티 배달"

> 1개 동네에서 도보/자전거 배달 + 공동주문 + 초저수수료

```
핵심 구조:
  점주 수수료: 0~2%
  소비자 배달비: 0~1,500원 (도보/자전거 배달)
  공동주문: 3명 이상 → 배달비 0원
  라이더: 도보/자전거 배달원 (초기) + 배달대행사 연동 (확장 시)
  수익원: 소비자 구독(월 4,900원) + 광고(장기) + 배달비 마진 + 선정산
```

**장점:**
- 초기 자본 매우 낮음 (1.5~3억원)
- 공동주문의 카카오톡 바이럴 = 무료 마케팅
- 도보 배달로 배달비 구조적 우위
- 배민이 따라하기 어려움 (전국 단위 시스템과 충돌)
- "우리 동네 배달앱" 정체성

**단점:**
- 확장할 때마다 cold start 반복
- 도보 배달 범위 제한 (반경 1~2km)
- 비/눈 올 때 도보 배달 한계
- 매우 작은 시장에서 시작

**적합한 시작 지역:** 대학가, 고밀도 주거지역 (성수, 연남, 합정 등)

---

### 후보 B: "B2B 기업 식대 플랫폼 → B2C 확장"

> 기업 점심 배달로 시작해서 개인 배달로 확장

```
핵심 구조:
  점주 수수료: 2~3%
  기업 고객: 월정액 SaaS (부서별 예산 관리, 세금계산서 자동)
  소비자 배달비: 기업 부담 또는 일반 배달비
  라이더: 배달대행사 연동
  수익원: 기업 SaaS 월정액 + 점주 수수료 2~3% + 광고
```

**장점:**
- Cold start 해결이 가장 쉬움 (기업 1곳 = 수백 명 수요)
- 기업 고객은 이탈률 낮음
- 객단가 높음 (단체 주문)
- 점심 시간 집중 → 라이더 효율 극대화
- "회사에서 쓰던 앱 → 집에서도" 자연 B2C 확장

**단점:**
- B2B 영업이 쉽지 않음 (의사결정 느림)
- 원래 비전("모든 이해당사자를 위한 플랫폼")과 다소 거리
- 점심 시간 외 수요 확보 과제
- 수수료 2~3%가 점주에게 매력적이지만 0%는 아님

**적합한 시작 지역:** 판교, 여의도, 강남 테헤란로 등 오피스 밀집 지역

---

### 후보 C: "투명 상생 배달" (하이브리드)

> 점주가 자발적으로 소비자를 데려오는 선순환 구조

```
핵심 구조:
  점주 수수료: 2% (배민 6.8% 대비 1/3)
  핵심 정책: "우리 플랫폼에서는 매장가 = 배달가" 장려
  소비자 혜택: 타 플랫폼 대비 음식값 5~15% 저렴 체감
  공동주문: 동일 건물/아파트 단지 묶음 → 배달비 분담
  라이더: 배달대행사 연동 + 투명 배차
  수익원: 수수료 2% + 소비자 구독 + 광고 + 선정산 + 배달비 마진
```

**핵심 메커니즘 - "점주가 영업사원이 되는 구조":**

```
1. 점주: 수수료 2%면 배민(6.8%) 대비 매출 500만원 기준 월 24만원 절감
2. 점주가 그 절감분 일부를 음식 가격에 반영 (매장가 동일 또는 5~10% 할인)
3. 점주가 자발적으로 "이 앱으로 주문하면 더 저렴합니다" 홍보
   - 매장 내 안내문, 배달 시 전단지 동봉, SNS 안내
4. 소비자가 실제로 가격 차이를 체감하고 전환
5. 주문량 증가 → 더 많은 점주 입점 → 가맹점 다양성 증가 → 더 많은 소비자
```

**왜 점주가 가격을 낮출 유인이 생기는가:**
- 수수료 절감분 전액을 가격 인하에 쓸 필요 없음 (일부만)
- 우리 플랫폼에서 가격을 낮추면 → 주문량 증가 → 총이익 증가
- 플랫폼이 "매장가 동일" 인증 뱃지 부여 → 소비자 신뢰 → 노출 우대
- 핵심: **강제가 아닌 인센티브** (가격 낮추면 노출 혜택, 안 낮춰도 OK)

**장점:**
- 소비자 전환의 가장 큰 장벽("왜 바꿔?")을 직접 해결 (음식값이 실제로 더 싸니까)
- 점주의 자발적 홍보로 CAC(고객획득비용) 대폭 절감
- 수수료 2%로 안정적 수익 확보
- 확장성이 후보 A보다 좋음 (도보 배달 제한 없음)
- 공동주문으로 배달비 추가 절감 가능

**단점:**
- 점주가 실제로 가격을 낮출지는 불확실 (핵심 리스크)
- 배민이 수수료 인하로 대응할 수 있음
- 수수료 2%는 0%가 아니라는 비판 가능
- 초기에 가맹점 다양성 확보 과제는 여전

**적합한 시작 지역:** 맛집 밀집 + 주거 밀집 지역 (성수, 합정, 연남, 망원 등)

## 최종 추천: A + C 결합

**"하이퍼로컬 투명 상생 배달"**

> 1개 동네에서 시작하되, 공동주문 + 투명 가격 + 점주 자발 홍보를 결합

```
구조:
  점주: 수수료 2% (배민 대비 1/3)
       → "매장가=배달가" 인증 시 노출 우대
       → 점주가 자발적으로 소비자에게 홍보

  소비자: 음식값 5~15% 저렴 (점주가 수수료 절감분 반영)
         + 공동주문(3명+) 시 배달비 0원
         + 구독(월 4,900원) 시 배달비 항상 할인

  라이더: 배달대행사 연동 (초기)
         + 투명 배차 기준 공개
         + 자전거/도보 배달 옵션 (근거리)

  수익: 수수료 2% + 배달비 마진 + 소비자 구독 + 광고(장기) + 선정산(장기)
```

**이 조합이 최선인 이유:**

1. **소비자 전환 동기가 명확**: "같은 음식이 여기서 더 싸다" + "공동주문하면 배달비 0원"
2. **점주 매력 극대**: 수수료 1/3 + 자발적 홍보 동기
3. **바이럴 내장**: 공동주문 = 카카오톡으로 친구/이웃 초대 = 무료 마케팅
4. **배민이 따라하기 어려움**: 수수료를 2%로 낮추면 배민 매출 70% 감소
5. **초기 자본 효율적**: 3~5억원으로 시작 가능
6. **확장 가능**: 동네별로 복제하며 성장

## 실행 로드맵

### Phase 0: 준비 (0~3개월)
- 법인 설립, MVP 개발
- 타겟 동네 선정 + 점주 50명 사전 인터뷰
- 팀: 개발 2~3명, 영업 1~2명, 운영 1명
- 자금: 1~2억원 (자비 또는 시드)

### Phase 1: 1개 동네 검증 (3~9개월)
- 점주 100개+ 입점 (수수료 0% → 6개월 후 2%)
- 공동주문 기능 출시 (카카오톡 공유)
- 일 주문: 100 → 500건
- 핵심 검증: "점주가 실제로 가격을 낮추는가?" "공동주문 바이럴이 작동하는가?"

### Phase 2: 3~5개 동네 확장 (9~18개월)
- 검증된 플레이북으로 동네 복제
- 소비자 구독 모델 도입
- 자체 배달(자전거) 일부 시작
- 일 주문: 500 → 3,000건
- Series A 투자 유치 (30~50억원)

### Phase 3: 수도권 확장 (18~36개월)
- 서울 주요 동네 20~30개 커버
- 광고 수익 본격화
- 금융 서비스(선정산) 도입
- 일 주문: 3,000 → 15,000건
- 흑자 전환 목표
