# 오픈 프로토콜 기반 배달비 구조적 절감 방안 탐색

> 작성일: 2026-03-23
> 배경: 오픈 프로토콜 + 중앙화된 품질 레이어 + 수수료 1~2% 모델에서
> 소비자 배달비 3,000원을 쿠팡이츠(0원)과 경쟁 가능한 수준으로 낮추는 방법

## 현재 배달비 원가 구조 (기준점)

### 소비자가 내는 배달비 3,000원의 분해

```
소비자 배달비 3,000원 → 라이더에게 실제로 가는 금액은?

현재 구조 (배달대행사 경유):
┌─────────────────────────────────────────────────────────┐
│ 소비자 배달비          3,000 ~ 4,000원                    │
│ + 점주 부담 배달비     1,500 ~ 2,900원                    │
│ = 총 배달 원가         4,500 ~ 6,900원                    │
│                                                         │
│ 분배:                                                    │
│  ├ 라이더 실수령       3,000 ~ 4,040원 (건당 평균)        │
│  ├ 배달대행 본사(바로고/부릉)  50 ~ 200원 (콜당)          │
│  ├ 지역 배달대행 지사         500 ~ 1,000원               │
│  └ 플랫폼(배민 등)           0 ~ 500원                    │
└─────────────────────────────────────────────────────────┘
```

### 핵심 수치 (2025년 기준)

| 항목 | 금액 | 출처 |
|------|------|------|
| 건당 평균 배달료 (라이더 수령) | 4,040원 (2025.02 기준, 전년 4,340원에서 하락) | 세계일보 |
| 배달대행 본사 수수료 | 건당 50~200원 (평균 80원) | 나무위키/뉴스토마토 |
| 지역 배달대행 지사 마진 | 건당 500~1,000원 | 시사인/나무위키 |
| 쿠팡이츠 점주부담 배달비 | 1,900~2,900원 | 쿠팡뉴스룸 |
| 쿠팡이츠 중개수수료 | 9.8% | 쿠팡뉴스룸 |
| 배민 중개수수료 | 6.8~7.8% | 한솥창업매거진 |
| 라이더 월 평균 수입 | 플랫폼 직속 320만, 대행사 280만 | 네이트뉴스 |
| 라이더 시간당 처리건수 (AI배차) | 3.2~3.3건 | 매일경제/전자신문 |
| 라이더 시간당 처리건수 (자율선택) | 3.6건 | 매일경제 |

## 1. 배달 오픈마켓 (경쟁 입찰) 메커니즘

### 개념

여러 배달대행사(바로고, 부릉, 생각대로, 뉴트랙 등)가 동일 주문에 대해 가격을 경쟁 입찰하고, 시스템이 최저가를 자동 선택하는 구조.

### 해외 레퍼런스: Ordering.co 모델

이미 글로벌에서 실현되고 있는 모델:
- Ordering.co는 Uber Direct, DoorDash Drive, Lalamove, Yango 등 복수 배달 파트너를 동시 연결
- 주문 발생 시 모든 파트너에게 실시간 견적 요청 → **지역/가격/배달시간 기준으로 자동 최적 선택**
- 하나가 실패하면 자동으로 다음 파트너에게 캐스케이드
- 셋업 비용 무료, 실시간 비용 투명하게 대시보드 제공

### 한국 적용 시 예상 효과

```
현재 구조:
  매장 → 1개 배달대행사 고정 계약 → 해당 지사 라이더만 배차
  → 경쟁 없음, 가격 협상력 약함

오픈마켓 구조:
  매장 → 오픈 프로토콜 → [바로고/부릉/생각대로/뉴트랙/개인라이더] 동시 입찰
  → 최저가 자동 선택

예상 절감:
  현재 건당 총 배달원가: 4,500 ~ 5,500원
  경쟁 입찰 시 절감: 10~20% (배달대행사간 마진 압축)
  절감 금액: 건당 450 ~ 1,100원
```

### 한국 유사 사례: 딜버(DEALVER)

- 2018년 출시된 배달대행 프로그램
- 복수 배달대행사를 비교할 수 있는 기능 제공
- 다만 완전한 실시간 경쟁 입찰까지는 아직 아님

### 실현 가능성 평가

| 항목 | 평가 |
|------|------|
| 기술 난이도 | 중 (API 연동 필요, Ordering.co가 이미 구현) |
| 절감 효과 | 건당 500~1,000원 |
| 위험요소 | 배달대행사가 담합하거나 연동 거부 가능 |
| 시간 | 3~6개월 (API 연동) |

## 2. 라이더 직접 연결로 중간자 제거

### 현재 다층 구조의 비용

```
현재 흐름:
  배달 플랫폼(배민)
    → 배달대행 본사(바로고/부릉): 건당 50~200원
      → 지역 배달대행 지사: 건당 500~1,000원 (라이더 관리, 영업, CS)
        → 라이더: 건당 3,000~4,040원

중간자 비용 합계: 건당 550~1,200원
```

### 오픈 프로토콜로 제거 가능한 비용

```
오픈 프로토콜 직접 연결:
  우리 플랫폼 → 직접 라이더 풀

  제거되는 비용:
  ├ 배달대행 본사 수수료: -80원
  ├ 지역 지사 마진: -500~1,000원
  └ 합계 절감: 건당 580~1,080원

  추가 발생 비용:
  ├ 라이더 모집/온보딩 비용: 인당 5~10만원 (1회성)
  ├ 라이더 관리 시스템 개발/운영: 월 2,000~5,000만원
  ├ CS/사고 처리: 건당 50~100원 (보험 포함)
  └ 배차 시스템 운영: 건당 20~50원

  순 절감액: 건당 410~930원
```

### 한국 프리랜서 라이더 직접 연결 현황

- **배민커넥트**: 배민이 직접 프리랜서 라이더를 모집/관리하는 모델. 이미 작동 중
- **쿠팡이츠 배달파트너**: 쿠팡이 직접 라이더 관리. 도보/자전거/오토바이 모두 지원
- **CallRider, Riderz**: 라이더 커뮤니티 앱. 직접 연결보다는 정보 공유 중심

### 현실적 판단

직접 라이더 풀을 운영하려면 **초기에 충분한 콜 볼륨이 필수**.
라이더는 "콜 안정성"이 최우선이므로, 일일 최소 배달 건수를 보장하지 못하면 이탈.

**단계적 접근**:
1. 초기: 바로고/부릉 API 연동 (중간자 비용 감수)
2. 성장기: 일부 지역에서 직접 라이더 풀 실험
3. 안정기: 직접 라이더 + 배달대행사 하이브리드 (경쟁 입찰)

## 3. AI 배차 최적화로 라이더 효율 극대화

### 현재 수준과 개선 여지

| 지표 | 현재 (AI 배차) | 현재 (자율 선택) | 이론적 최적 |
|------|---------------|-----------------|------------|
| 시간당 처리 건수 | 3.2~3.3건 | 3.6건 | 4.5~5건 |
| 건당 소요시간 | 18~19분 | 17분 | 12~13분 |
| 라이더 시간당 수입 | 13,000원 | 14,500원 | 20,000원 |

### 배민의 AI 배차 성과 (검증된 수치)

- 전체 배달 시간 **26% 감소** (전자신문 보도)
- 음식점 픽업까지 소요 시간 **43% 감소**
- 고객 전달 시간 **17% 감소**
- 라이더 사고율 **32~47% 감소**
- 부릉(메쉬코리아) AI 추천배차: 라이더 생산성 **11% 증가**

### 배달비에 미치는 영향 계산

```
현재: 라이더 시간당 3.3건 × 건당 4,040원 = 시간당 수입 13,332원
목표: 라이더 시간당 4.5건 × 건당 X원 = 시간당 수입 13,332원 (동일 수입 유지)

X = 13,332 ÷ 4.5 = 2,963원

→ AI 최적화만으로 건당 배달비를 4,040원 → 2,963원으로 낮출 수 있는 이론적 여지
→ 절감: 건당 약 1,077원 (라이더 수입 동일 유지 전제)
```

### 하지만 현실적 한계

- 배민/쿠팡이츠도 이미 AI 배차를 강하게 도입 중
- 라이더들의 불만: "AI 배차대로 했더니 시간 더 걸리고 수입은 줄어" (매일노동뉴스)
- AI가 "산을 뚫고 12분 만에 가라"고 지시하는 등 현실 반영 부족 (한국일보)
- **우리가 배민보다 나은 AI를 만들기는 어려움** → 차별화 요소가 아님

### 현실적 절감 가능 금액

기존 배달대행사 대비 AI 개선으로 **건당 300~500원** 추가 절감이 현실적.
(배민 수준의 AI를 전제하되, 하이퍼로컬 특화 최적화 적용)

## 4. 하이퍼로컬 범위 제한으로 배달 효율화

### 배달 반경에 따른 비용 변화

```
일반 배달 (반경 3~5km):
  - 오토바이 기준 왕복 15~25분
  - 건당 배달비: 4,000~5,000원
  - 시간당 처리: 2.5~3건

하이퍼로컬 (반경 1~2km):
  - 전기자전거 기준 왕복 8~15분
  - 건당 배달비: 2,000~3,000원
  - 시간당 처리: 4~6건 (배민/쿠팡 자전거 라이더 실측)

초하이퍼로컬 (반경 500m~1km):
  - 도보 기준 왕복 10~20분
  - 건당 배달비: 1,500~2,500원
  - 시간당 처리: 3~4건
```

### 도보/자전거 배달 실측 데이터

| 수단 | 시간당 처리 | 건당 수입 | 일일 수입 (8시간) | 출처 |
|------|-----------|----------|-----------------|------|
| 도보 | 2.5~3건 | 2,500~3,000원 | ~6만원 (20~22건) | 클리앙 |
| 전기자전거 | 4~6건 | 2,500~3,500원 | ~10만원 | 배민커넥트 실측 |
| 오토바이 | 3~4건 | 4,000~5,000원 | ~12만원 | 배달대행 평균 |

### 하이퍼로컬의 핵심 장점

1. **전기자전거/도보 가능** → 차량 유지비 대폭 절감
2. **픽업 대기시간 감소** → 가맹점이 가까우니 음식 완성 타이밍 맞추기 쉬움
3. **묶음배달 효율 극대화** → 같은 동네 2~3건을 한번에
4. **악천후 리스크 감소** → 1km 이내라 폭우/폭설에도 배달 가능

### 건당 비용 계산

```
하이퍼로컬 전기자전거 배달 원가:

라이더 비용:
  시간당 목표 수입: 12,000원 (시급 환산)
  시간당 처리 건수: 5건
  → 건당 라이더 비용: 2,400원

플랫폼 비용:
  배차 시스템: 50원
  CS/보험: 100원
  → 건당 플랫폼 비용: 150원

총 건당 배달 원가: 2,550원
```

## 5. 배달 수단 혁신: 전기자전거 vs 오토바이

### 월간 유지비 비교

| 비용 항목 | 오토바이 (125cc) | 전기자전거 |
|-----------|-----------------|-----------|
| 차량 구입/감가 | 월 15~20만 (200만원/12개월) | 월 8~12만 (100~150만원/12개월) |
| 연료/충전 | 월 15~25만 (휘발유) | 월 0.5~1만 (전기) |
| 보험료 | 월 8.5~22만 (연 103~265만원) | 월 0.5~2만 (자전거 보험) |
| 수리/정비 | 월 5~10만 | 월 3~5만 (브레이크, 타이어) |
| 렌트 옵션 | 월 25~35만 | 일 5,000원 (월 15만원) |
| **월 합계** | **43.5~77만원** | **12~20만원** |
| **건당 환산 (월 600건)** | **725~1,283원** | **200~333원** |

### 핵심 절감 포인트

```
오토바이 → 전기자전거 전환 시:
  건당 유지비 절감: 525~950원

  단, 배달 반경 제한: 3~5km → 1~2km
  단, 배달 건수 제한: 시간당 3~4건 → 4~6건 (짧은 거리로 보상)
  단, 무거운 음식 한계: 대량 주문 불가
```

### 전기자전거 배달의 현실

- 배민커넥트 전기자전거 라이더: 2시간에 3만 7천원 수입 실측 (뉴데일리)
- 바이크뱅크: 일 5,000원 렌트비로 전기자전거+라이더보험 패키지 제공
- 전기자전거 평속 20km/h → 1km 배달 편도 3분, 왕복 10분 (픽업/하차 포함)
- 충전 1회 40~80km 주행 가능 → 하루 배달 충분

## 6. 묶음배달(배치배달)의 비용 절감 효과

### 현재 묶음배달 현황

| 서비스 | 묶음 건수 | 배달비 절감 | 배달시간 증가 |
|--------|----------|-----------|-------------|
| 배민 알뜰배달 | 2건 | -1,000원 | +10~20분 |
| 쿠팡이츠 세이브배달 | 최대 2건 | -1,000원 | +10~15분 |
| B2B 오피스 묶음 (벤디스) | 5~20건 | 배달비 0원 | 시간 지정 |

### 묶음배달의 건당 비용 산출

```
단건 배달:
  라이더 1명 → 1건 → 건당 4,040원

2건 묶음:
  라이더 1명 → 2건 → 건당 약 2,500원 (총 5,000원)
  절감: 건당 1,540원 (38% 감소)

3건 묶음 (같은 동네):
  라이더 1명 → 3건 → 건당 약 1,800원 (총 5,400원)
  절감: 건당 2,240원 (55% 감소)

5건 묶음 (B2B 오피스):
  라이더 1명 → 5건 → 건당 약 1,000원 (총 5,000원)
  절감: 건당 3,040원 (75% 감소)
```

### 하이퍼로컬 + 묶음의 시너지

```
하이퍼로컬 (반경 1km) + 2건 묶음:
  전기자전거 라이더 → 2건 동시 픽업 → 같은 방향 배달
  소요시간: 15~20분 (2건)
  건당 비용: 1,500~2,000원

하이퍼로컬 + 시간대 묶음 (점심/저녁 피크):
  3건이 모이면 출발 → 건당 1,200~1,500원
```

## 7. 쿠팡이츠 "배달비 0원"의 실체와 시사점

### 쿠팡이츠 0원의 구조

```
쿠팡이츠 스마트요금제:
  - 소비자 배달비: 0원 (묶음배달 한정)
  - 점주 부담 배달비: 1,900~2,900원
  - 점주 중개수수료: 9.8%
  - 라이더 지급 배달비: ~4,000원

  적자 구조:
  점주부담 2,900원 + 수수료수입 ~1,000원(주문금액 10,000원 × 9.8%) = 3,900원
  라이더 지급: ~4,000원
  → 건당 ~100원 적자 (+ 플랫폼 운영비)
```

### 시사점

1. **쿠팡이츠는 적자를 감수하며 시장점유율 확보 중** (5년간 흑자 없음)
2. **실질적으로 점주에게 배달비를 전가** (수수료 9.8% + 배달비 2,900원 = 매출의 ~20%)
3. **단건배달은 여전히 유료** (0원은 묶음배달만)
4. **소비자가 "0원"에 중독되면 유료 전환 시 이탈 위험**

### 우리의 대응

쿠팡처럼 적자를 감수할 수 없으므로, **구조적으로 배달비를 낮추는 것**이 핵심.
"0원"이 아니라 **"합리적으로 낮은 배달비 + 투명한 가격"**이 전략.

## 10. 핵심 리스크와 대응

### 리스크 1: "하이퍼로컬이면 가맹점이 적다"
- **대응**: 1개 동(洞) 단위로 집중 가맹. 동네 전체 음식점 80% 이상 확보 후 런칭
- 한 동네에 음식점 50~100개면 소비자 선택 다양성 충분

### 리스크 2: "묶음배달은 배달이 느리다"
- **대응**: 하이퍼로컬이므로 묶음이어도 20~25분 내 배달 가능
- 반경 1km 내 2건 묶음 = 일반 배달 3km 단건보다 빠를 수 있음

### 리스크 3: "전기자전거 라이더가 부족하다"
- **대응**: 바이크뱅크 파트너십으로 렌탈 제공 (일 5,000원)
- 대학생/시니어 등 오토바이 면허 없는 인력 활용 가능
- 진입장벽 낮음 → 라이더 풀 확대 용이

### 리스크 4: "쿠팡이츠가 0원으로 공격하면?"
- **대응**: 쿠팡이츠 0원은 적자 모델(5년간 흑자 없음), 언젠가 유료 전환 필수
- 우리는 **구조적으로** 원가가 낮으므로 지속 가능
- "투명한 1,000원" vs "숨겨진 전가 0원" → 신뢰 기반 차별화

### 배달비 3,000원 → 1,000원은 6개월 내 가능

| 구간 | 소비자 배달비 | 원가 | 커버 방법 |
