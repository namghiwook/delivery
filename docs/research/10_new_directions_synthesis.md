# 새로운 방향 종합 분석 - 4개 에이전트 교차 분석

> 작성일: 2026-03-23
> 기존 삼각 딜레마를 우회하는 접근법 탐색 결과

## 삼각 딜레마 재확인

수수료 절감(%) < 배달비(고정 원) → 같은 게임 규칙으로는 해결 불가.
보조금도 1년 만에 650억→0원 삭감 → 지속 불가능.

**핵심 원칙 (4개 에이전트 공통 결론):**
> "기존 강자가 따라하면 자기 사업이 무너지는 모델을 만들어라"

배민 매출 4.3조원의 핵심은 건당 수수료(7.8%) + 광고.
"건당 수수료 0원"을 선언하면 배민은 연 수천억 수익을 포기해야 → 따라할 수 없음.

## 게임 체인저 5가지 패턴

| 패턴 | 사례 | 배달 시장 적용 |
|------|------|--------------|
| 따라하면 자기 잠식 | 코스트코(마진 3%), 토스(중립 비교) | 건당 수수료 0원 → 배민 수익 붕괴 |
| 무시당하는 세그먼트 | 핀둬둬(3선 도시), 당근(소액 중고) | 배민에 없는 가게, B2B 점심 |
| 니즈 재정의 | 에어비앤비(살아보는 여행), 스포티파이(소유→접근) | 배달→경험/구독/커뮤니티 |
| 비용 구조 변경 | IKEA(고객이 조립), Southwest(기내식 제거) | 묶음배달, 도보배달, 중간자 제거 |
| 네트워크 효과 방어벽 | 핀둬둬(친구 초대), 당근(동네 커뮤니티) | 동네 기반 주문 커뮤니티 |

## 4개 에이전트에서 반복 등장한 유망 방향 3가지

### 방향 1: "점주 직접 주문 인프라" (SaaS)

**배달앱에 맞서 싸우는 게 아니라, 점주가 배달앱 없이도 장사할 수 있게 해주는 도구**

```
현재: 점주 → [배민 수수료 7.8% + 광고] → 소비자
우리: 점주 → [네이버/카카오 직접주문 0.8~1.1%] + [배달대행사 연동] → 소비자
     우리 역할: 이 두 개를 심리스하게 연결하는 SaaS
```

- 네이버 직접주문 수수료: 0.8~1.1% (배민 7.8% 대비 1/7)
- 배달은 바로고/부릉에 직접 연결
- 점주용 CRM(단골 관리) + B2B 점심 채널 + 매출 분석
- 월정액 5~10만원
- **배민이 따라할 수 없는 이유**: 배민이 "점주가 배민 없이 주문받게 해준다"를 만들 리 없음

근거:
- 배민 포장 수수료 6.8% 도입(2025.4) 이후 "네이버 주문으로 갈아타자" 움직임 확산
- 교촌치킨 자체앱 연 650억 매출, BBQ 멤버십 30만명 → 프랜차이즈는 이미 탈배민 시작
- 소상공인 전용 솔루션은 아직 빈 공간

### 방향 2: "B2B 오피스 점심 배달"

**묶음배달로 배달비 자체를 제거**

```
벤디스 "배달대장" (이미 검증됨):
- 오전 9:30까지 주문 → 점심 전 오피스 빌딩으로 묶음 배달
- 배달비 0원, 가입비 0원
- 평균 메뉴가 7,000원
- 점주: 유휴시간(오전) 매출 증가
- 라이더: 같은 건물 다건 배달 → 효율 극대화
```

- 기업 식대 시장: 연 20조원
- 2026년 정부 '든든한 한끼' 정책: 직장인 점심값 20% 환급(월 최대 4만원)
- 배민/쿠팡은 B2C에 최적화 → B2B 기업 영업(세금계산서, 예산관리)은 다른 역량

### 방향 3: "동네 구독형 배달" (코스트코 모델)

**배달비가 "보이지 않는" 구조**

```
소비자: 월 9,900원 구독 → 동네 가맹점 배달비 무료 + 5~10% 할인
점주: 월 15~30만원 회비 → 건당 수수료 0원
배달: 동네 묶음배달 (같은 동네 3건 모이면 출발)
```

- 코스트코처럼 "마진은 포기, 회비로 번다"
- 배달비가 구독료에 포함 → 소비자가 배달비를 별도로 인식하지 않음
- 배민이 따라할 수 없음: 건당 수수료 0원은 배민 매출의 핵심을 포기하는 것

## 방향별 비교

| 기준 | 1. 점주 SaaS | 2. B2B 점심 | 3. 동네 구독 |
|------|-------------|------------|------------|
| 삼각 딜레마 해결 | 배달비 문제 회피 (점주 채널) | 묶음배달로 배달비 제거 | 구독료에 포함 |
| 검증 수준 | 부분 (네이버 직접주문 확산 중) | 높음 (벤디스 검증) | 낮음 (미검증) |
| 배민 대응 불가 이유 | 자기 존재 부정 | B2B는 다른 역량 | 건당 수수료 포기 |
| 초기 자본 | 낮음 (2~3억) | 중간 (5억) | 중간 (5억) |
| 확장성 | 높음 (SaaS 복제) | 높음 (오피스 지역 복제) | 낮음 (동네별 반복) |
| 비전 부합도 | ★★★★★ | ★★★★☆ | ★★★★★ |

## 추가 발견

### 가치사슬 비효율 (수익 기회)
- 다층 중개 구조: 배달대행 본사(80원/콜)+지사 제거 시 건당 600~1,000원 절감
- 광고비 제로섬 게임: 배민 매출의 상당 부분이 비생산적 광고비 경쟁에서 발생
- 배달대행사 대부분 적자: 바로고 -166억, 부릉 적자 → 시장 재편 중
- 라이더 실질 시급 7,864원 (최저임금 미달) → 사회적 문제

### 실패 확인된 모델 (피해야 할 것)
- ONDC 오픈 프로토콜: 인도에서 음식 배달은 실패 (주문 650만→460만건 급감)
- 공유주방: 위쿡 서비스 중단, 80% 중도포기율
- 블록체인/Web3 배달: 실용 사례 전무
